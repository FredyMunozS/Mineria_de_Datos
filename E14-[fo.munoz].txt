LUIS (Language Understanding Intelligent Service)

Es un componente de Microft Azure que básicamente funciona como un API que está alojada en la nube, utiliza diferentes técnicas de machine learning para predecir el significado global del texto introducido en lenguaje natural por un usuario.
Existen múltiples aplicaciones de cliente que pueden consumir los servicios de LUIS, estas son aplicaciones conversacionales que interactúan con usuarios en forma de lenguaje natural para completar tareas determinadas. Algunos ejemplos de aplicaciones de cliente son aplicaciones de redes sociales, chat bots y aplicaciones de escritorio habilitadas para voz.  
La interacción de las aplicaciones de usuario con LUIS es sencilla, por un lado la aplicación envía expresiones de texto al API en sus propias palabras,  LUIS aplica el modelo entrenado al texto en lenguaje natural para proporcionar reconocimiento inteligente sobre la entrada del usuario. LUIS devuelve una respuesta con formato JSON con una intención principal, "HRContact" ("Contacto de recursos humanos"). La respuesta en formato JSON del punto de conexión contiene como mínimo la expresión de consulta y la intención con mayor puntuación. También puede extraer datos, como la entidad Tipo de contacto. Finalmente la aplicación cliente usa la respuesta JSON para tomar decisiones acerca de cómo responder a las solicitudes del usuario. Estas decisiones pueden incluir algunos árboles de decisión en el código de Bot Framework y llamadas a otros servicios.
LUIS tiene muchos modelos de dominio creados previamente, entre los que se incluyen intenciones, expresiones y entidades creadas previamente, adicionalmente ofrece varias maneras de identificar sus propias intenciones y entidades personalizadas, como las entidades de aprendizaje automático, las entidades específicas o literales y una combinación de aprendizaje automático y literal. Una vez que un modelo de LUIS está publicado y recibe expresiones de usuario reales, LUIS ofrece varios métodos para mejorar la precisión de la predicción.
